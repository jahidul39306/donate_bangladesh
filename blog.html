What is the Document Object Model (DOM)?

DOM is the data representation of the HTML codes. To manipulate or access the HTML code dynamically via programming language, DOM is used.

How do you select an element from the DOM?

There are many ways to select an element from the DOM, but the popular one is 'document.getElementById('id')'. We can also use 'querySelector()', 'querySelectorAll('.class-name')', 'getElementsByTagName('h1')', and 'getElementsByClassName('class-name')'

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a design pattern to manage and handle events. It is used in parent element to listen event of the child element. When there are similar child element, we can just put event listener to the parent element and can listen to the target child element by 'event.target'.

How do you manipulate an element's attributes and styles using the DOM?

We can use 'setAttribute', 'getAttribute', 'removeAttribute' method's to manipulate attribute. To manipulate style we can use 'element.style.style-property'. If we are using any css framework, we can use 'element.classList.add('style-property')' and 'element.classList.remove('style-property')'
